## Referências do projeto de classificações

Este diretório reúne as principais referências conceituais e normativas utilizadas no projeto do **Classificador de Natureza de Receita do Estado de Minas Gerais**.

Cada referência listada neste diretório segue uma estrutura básica para facilitar sua identificação e uso no projeto. Os principais campos de um registro de referência são:

- **Título e versão**: nome da referência, incluindo o número da versão (quando aplicável).
- **Fonte/autoria**: o órgão ou entidade responsável pela publicação (ex.: UNECE/ONU).
- **Tipo**: define se a referência é um modelo conceitual, uma diretriz, um manual, etc.
- **Escopo principal**: breve descrição do foco principal da referência e dos tópicos abordados.
- **Relação com o projeto**: explica como a referência fundamenta decisões ou processos específicos do projeto do classificador.
- **Arquivos locais**: lista os arquivos disponíveis neste diretório (em formatos como PDF, Markdown ou links), facilitando o acesso direto ao conteúdo.

### Referências principais

- **GSIM: Statistical Classifications Model v1.1 (UNECE/ONU, 2015)**  
  - **Tipo**: modelo conceitual / modelo de informação.  
  - **Escopo principal**: define os objetos e atributos necessários para modelar classificações estatísticas (séries, classificações, versões, variantes, itens, níveis etc.), além de princípios de governança e de interoperabilidade.  
  - **Relação com o projeto**: base conceitual para o desenho do banco de dados de classificações de receita; fundamenta o tratamento de versões, atualizações, variantes e classificação flutuante.  
  - **Arquivos locais**:  
    - Notas em Markdown: `gsim_v1-1_unece-ONU_2015_notes.md`  
    - Texto completo: `gsim_v1-1_unece-ONU_2015.md`  
    - PDF: `gsim_v1-1_unece-ONU_2015.pdf`

- **GSIM v2.0 – Generic Statistical Information Model (UNECE/ONU, 2023)**  
  - **Tipo**: modelo conceitual geral para informação estatística.  
  - **Escopo principal**: versão 2.0 do GSIM, com reestruturação em cinco grupos principais (Base, Business, Concept, Exchange, Structure) e refinamentos em diversos objetos, incluindo metadados referenciais e eventos de mudança.  
  - **Relação com o projeto**: usado como referência de evolução do padrão; ajuda a alinhar o modelo de classificações de receita com o estado atual do GSIM, especialmente no grupo Concept.  
  - **Arquivos locais**:  
    - Notas em Markdown: `gsim_v2-0_unece-ONU_2023.md`

- **Best Practice Guidelines for Developing International Statistical Classifications (UNSD/ONU, 2022)**  
  - **Tipo**: diretrizes de boas práticas.  
  - **Escopo principal**: orientações sobre governança, desenvolvimento, manutenção, revisões e disseminação de classificações estatísticas, com foco em classificações internacionais.  
  - **Relação com o projeto**: fundamenta processos de revisão, consulta, documentação de mudanças, definição de versões e atualização contínua do classificador de receita.  
  - **Arquivos locais**:  
    - Notas em Markdown: `best-practice-guidelines-international-classification_usd-ONU_2022.md`  
    - Link: `best-practice-guidelines-international-classification_usd-ONU_2022.url`  
    - PDF: `best-practice-guidelines-international-classification_usd-ONU_2022.pdf`
